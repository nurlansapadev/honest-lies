# Артефакты

Медиафайлы хранятся локально на диске D, не в этом репозитории.

## Структура на D:\YouTube-Studio\

```
D:\YouTube-Studio\
└── ep001-[название]\
    ├── Исходники\     # архивное видео, фото, аудио от актёра
    ├── Монтаж\        # проекты DaVinci / Premiere
    └── Результат\     # финальный mp4, опубликованная версия
```
