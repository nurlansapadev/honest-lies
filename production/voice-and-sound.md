## Движок и модель

- **Единственная модель — Eleven v3.** 
- v3 варьируется между генерациями — **норма: 2–3 дубля на блок, выбираем лучший.**
- Генерируем **короткими блоками** (3–6 предложений), не весь сценарий разом.

## Голос Arny — зафиксирован навсегда

**Voice ID: `2NLhtWk1HfS8hMiHqknR`**

Характер: низкий, сухой, неспешный американец ~60 лет, нью-йоркский акцент.
Deadpan. Говорит со зрителем как с равным. Не диктор, не трейлер — человек рассказывает историю.

## Базовые настройки (едины для всех видео)

| Параметр | Значение | Почему |
|----------|----------|--------|
| **Model** | Eleven v3 | единственная |
| **Stability** | **Natural** | баланс референса и отзывчивости на теги |
| **Similarity** | **~85%** | узнаваемость голоса от видео к видео |
| **Style** | **низкий** | deadpan — сдержанность, не игра |
| **Speed** | **чуть медленнее** | неспешность Arny |

> Robust — слишком игнорирует теги. Creative — галлюцинирует. Natural — наш.

## Четыре регистра — оттенки ОДНОГО голоса

Не разные люди. Один Arny, разный градус.

| Регистр | Где | Теги |
|---------|-----|------|
| **Ровный репортёр** | такты 1–2 | почти без тегов |
| **Впечатлённый профи** | такт 4 | `[admiringly]` точечно |
| **Деадпан-скептик** | такты 5–6 | `[sarcastic]` на иронии |
| **Маска снята** | такт 7 | `[thoughtful]` один раз |

## Audio Tags — полные правила (по документации v3)

### Разрешённый набор для Arny
```
[sarcastic]        ← ирония и deadpan
[thoughtful]       ← финал, маска снята
[sighs]            ← усталость от вранья (редко)
[short pause]      ← микропауза внутри фразы
[long pause]       ← пауза между блоками
[admiringly]       ← восторг к мастерству операции
[whispers]         ← особый акцент (очень редко)
[strong New York accent]  ← усилитель акцента (тест)
```

### Запрещено для Arny
```
[excited] [laughs] [crying] [shouting] [woo]
[happy] [angry] — буйные эмоции не в характере
```

### Дисциплина
- Тег ставится **перед** фразой: `[sarcastic] Convenient.`
- Максимум **1–2 тега на абзац** — deadpan живёт сдержанностью
- Теги работают по-разному на разных голосах — **всегда тестировать перед продакшном**
- Кнопка **Enhance** в UI добавляет теги автоматически — использовать как черновик, потом чистить вручную

## Паузы (v3 — только так, НЕ SSML)

> ⚠️ **v3 НЕ поддерживает `<break time="..."/>` (SSML).** Только:

| Приём | Эффект | Когда |
|-------|--------|-------|
| `…` (многоточие) | короткая пауза, вес | внутри фразы |
| `[short pause]` | ~0.5 сек | между предложениями |
| `[long pause]` | ~1–1.5 сек | после ударной фразы |
| Заглавные буквы | ударение на слове | ключевые слова |
| Тире `—` | микрозаминка | внутри предложения |

**Пуговицы тактов:** `[long pause]` после ударной строки → монтаж тишины.

## Акцент (экспериментально)

Тег `[strong New York accent]` — ставить перед фразой, где хочется усилить.
Тестировать отдельно: на этом голосе может быть нестабильно.
Если галлюцинирует — убрать, голос сам по себе держит акцент.

## Рабочий процесс (для каждого эпизода)

**Принцип: один такт = один кусок текста.** Не построчно. Берёшь целый такт, копируешь в ElevenLabs, генеришь. Теги тональности встроены прямо в поток текста по месту — отдельно ничего сводить не надо.

1. Открываешь файл такта (напр. `sound-ep-nur-01-t1.md`) — там готовый цельный текст с уже встроенными тегами и фонетикой.
2. Копируешь **весь кусок целиком** в ElevenLabs Studio (одна глава = один такт).
3. Генеришь, слушаешь, при необходимости — дубль.
4. Длинный такт (>2500 знаков — лимит генерации) заранее разбит на части: `-p1`, `-p2`. Каждая часть = отдельный кусок.
5. Имена собственные и фонетика уже вшиты в текст — отдельный файл не нужен.

**Теги — минимум, только на ключевых фразах**, встроены по месту (`[sarcastic] Convenient.`). Деадпан живёт сдержанностью.

**Паузы внутри текста:** `…` и `[short pause]`. Между тактами — тишина на монтаже, не в движке.

**Файлы тактов:** `sound-ep-NNN-tN.md` (такт целиком) или `sound-ep-NNN-tN-pM.md` (если такт длинный, part M).
